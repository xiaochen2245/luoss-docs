## 申请账号

使用平台前需要先申请账号。

::: warning 资源限制
每个用户/项目最多只能使用 **128 张 910C** 显卡。

- 集群资源紧张时，算卡分配额度无法完全保证：个人申请最多保证 **4 卡**，项目申请保障申请额度，多余资源会被驱逐
- 集群资源充足时，可以使用超出额度的算卡
:::

请使用**河套学院邮箱**发送邮件至 **hpc@slai.edu.cn**，邮件模板如下：

```
主题：平台账号申请 - [您的姓名]

您好，

我希望申请调度平台的访问账号，以下是资源需求信息：

- 邮箱：[填写您的邮箱地址]
- 所属科研中心（部门）：[填写所在科研中心或部门名称]
- 使用时间：[填写截止日期，如 2025-12-31]
- 用途说明：[简要描述使用目的]
- 关联项目：[如涉及具体项目，请填写项目名称，否则填"无"]

谢谢！
```

提交申请后，管理员会审核并通过邮件将账号信息发送给您。

## 前置要求

在开始之前，请确保您已经：
1. 拥有平台的访问账号（如尚未申请，请参考上方 [申请账号](#申请账号)）
2. 了解基本的命令行操作
3. 熟悉 Docker 镜像的基本概念
