# 项目管理

项目管理功能支持团队协作，项目管理员可以邀请成员、分配资源配额，成员可以在项目内提交训练任务。

<FeatureBadge status="stable" />

## 概述

### 角色权限

| 角色 | 权限 |
|------|------|
| **项目管理员** | 邀请成员、审批申请、分配配额、管理项目设置、查看所有作业 |
| **项目成员** | 查看项目作业、提交训练任务、使用分配的 NPU 配额 |

### 功能一览

| 功能 | 管理员 | 成员 |
|------|--------|------|
| 查看项目列表 | ✅ | ✅ |
| 提交训练任务 | ✅ | ✅ |
| 查看项目作业 | ✅ | ✅ |
| 邀请成员 | ✅ | ❌ |
| 审批加入申请 | ✅ | ❌ |
| 分配成员配额 | ✅ | ❌ |
| 项目设置 | ✅ | ❌ |

## 我的项目

点击左侧菜单 **我的项目** 进入项目列表页面。

### 项目卡片信息

每个项目卡片显示：

| 信息 | 说明 |
|------|------|
| 项目名称 | 项目显示名称 |
| 描述 | 项目简介 |
| 角色 | 管理员/成员 |
| 队列 | 关联的 Volcano 队列 |
| NPU 优先额度 | 项目可用的 NPU 资源 |

### 项目操作

| 操作 | 说明 |
|------|------|
| 查看作业 | 进入项目作业列表 |
| 成员管理 | 管理项目成员（仅管理员） |
| 项目设置 | 修改项目配置（仅管理员） |

## 加入项目

### 通过邀请链接

1. 获取项目管理员分享的邀请链接（格式：`/join?code=XXXXXXXX`）
2. 打开链接或访问 **加入项目** 页面
3. 输入 8 位邀请码
4. 点击 **申请加入**
5. 等待管理员审批

::: info 审批流程
- 提交申请后，状态显示为"待审批"
- 管理员批准后，您将成为项目成员
- 如果已经是项目成员，会直接提示已加入
:::

### 邀请码说明

| 状态 | 说明 |
|------|------|
| 有效 | 可以使用邀请码申请加入 |
| 已过期 | 需要管理员重新生成 |
| 已使用 | 邀请码可多次使用，直到管理员重新生成 |

## 项目管理（管理员）

### 生成邀请码

1. 进入 **我的项目**
2. 点击项目卡片的 **成员管理**
3. 在 **成员邀请** 区域点击 **生成加入码**
4. 分享邀请码或邀请链接给新成员

### 审批加入申请

1. 进入项目 **成员管理**
2. 查看 **待审批申请** 列表
3. 对于每个申请：
   - 点击 **批准** 接受成员加入
   - 点击 **拒绝** 拒绝申请

::: tip 提示
有待审批申请时，项目卡片的角色标签会显示红色角标提示数量。
:::

### 分配成员配额

项目管理员可以为每个成员分配 NPU 配额：

1. 进入项目 **成员管理**
2. 在成员列表中找到目标成员
3. 点击 NPU 配额列的 **调整** 按钮
4. 输入分配的 NPU 数量
5. 点击 **确定**

#### 配额分配规则

| 规则 | 说明 |
|------|------|
| 配额为 0 | 成员无法提交训练任务 |
| 管理员配额 | 管理员拥有无限配额 |
| 超额分配 | 总分配配额可以超过项目配额（允许超卖） |
| 已使用配额 | 调整配额不会清除已使用的配额 |

### NPU 配额分配概览

在成员管理页面，管理员可以看到：

| 统计项 | 说明 |
|--------|------|
| 项目总配额 | 项目拥有的 NPU 总量 |
| 已分配 | 已分配给成员的 NPU 总量 |
| 已使用 | 成员实际正在使用的 NPU |
| 可分配 | 剩余可分配的 NPU |

### 移除成员

1. 进入项目 **成员管理**
2. 在成员列表中找到目标成员
3. 点击 **移除** 按钮
4. 确认移除

::: warning 注意
- 不能移除项目管理员
- 移除成员后，该成员的作业会继续运行直到完成
- 成员数据（如训练模型）不会被删除
:::

## 项目设置

项目管理员可以修改项目设置：

1. 进入 **我的项目**
2. 点击项目卡片的 **项目设置**
3. 修改项目信息

### 可修改项

| 设置项 | 说明 |
|--------|------|
| 显示名称 | 项目显示名称 |
| 描述 | 项目简介 |

## 项目作业

### 查看项目作业

1. 进入 **我的项目**
2. 点击项目卡片的 **查看作业** 图标
3. 查看该项目下的所有训练任务

### 提交项目作业

在创建训练任务时，选择项目关联的队列即可使用项目资源。

::: info 配额限制
- 成员提交的任务使用的 NPU 不能超过分配的配额
- 管理员没有配额限制
:::

## 平台管理员

平台管理员拥有额外权限：

### 项目管理入口

1. 点击左侧菜单 **管理员指南** → **项目管理**
2. 或在 **我的项目** 页面点击 **管理项目** 按钮

### 管理功能

| 功能 | 说明 |
|------|------|
| 创建项目 | 创建新项目并指定管理员 |
| 编辑项目 | 修改项目配置、队列关联 |
| 删除项目 | 删除项目（需确认） |
| 分配项目管理员 | 指定或更换项目管理员 |

## 最佳实践

### 1. 项目组织

- 按**团队**或**研究方向**划分项目
- 为项目设置清晰的描述和命名规范
- 定期清理不再活跃的成员

### 2. 配额管理

- 根据成员实际需求分配配额
- 避免过度集中分配给少数成员
- 定期审查配额使用情况

### 3. 成员管理

- 及时审批加入申请
- 定期审查成员列表
- 为新成员提供必要的使用指导

### 4. 安全建议

- 定期重新生成邀请码
- 不要公开分享邀请链接
- 及时移除离职或不再参与的成员

## 常见问题

### Q: 如何成为项目管理员？

A: 项目管理员由平台管理员指定。如果您需要管理员权限，请联系平台管理员。

### Q: 配额用完了怎么办？

A: 联系项目管理员申请增加配额，或等待现有任务完成后释放配额。

### Q: 被移除后还能重新加入吗？

A: 可以。需要项目管理员重新发送邀请，您重新申请并等待审批。

### Q: 邀请码泄露了怎么办？

A: 项目管理员可以在成员管理页面点击"重新生成加入码"，旧邀请码将立即失效。
