# 我的使用统计

使用统计页面展示您的资源使用情况和历史记录。

## 功能概述

| 功能 | 说明 |
|------|------|
| **配额概览** | 当前配额和使用情况 |
| **使用趋势** | 资源使用趋势图 |
| **历史记录** | 历史使用明细 |

## 配额概览

### 资源配额

| 资源 | 说明 |
|------|------|
| CPU | CPU 配额和已用量 |
| 内存 | 内存配额和已用量 |
| NPU | NPU 配额和已用量 |

### 使用率图表

以图表形式展示各项资源的使用率：

- 柱状图显示已用/总量
- 百分比显示使用率

## 使用趋势

### 时间范围

可选择查看不同时间范围：

- 今日
- 近 7 天
- 近 30 天

### 趋势图

- CPU 使用趋势
- 内存使用趋势
- NPU 使用趋势

## 历史记录

### 使用明细

| 字段 | 说明 |
|------|------|
| 时间 | 使用时间 |
| 资源类型 | CPU/内存/NPU |
| 使用量 | 使用数量 |
| 来源 | 环境或任务 |

## 数据导出

支持导出使用统计数据：

1. 点击 **导出** 按钮
2. 选择时间范围
3. 下载 CSV 文件

## 优化建议

1. **关注使用峰值**：了解资源使用高峰期
2. **合理规划**：根据趋势规划资源需求
3. **及时释放**：释放不使用的资源
