# 仪表盘

仪表盘是您登录后的首页，展示资源使用概览和重要信息。

## 功能概述

仪表盘提供以下信息：

| 区域 | 说明 |
|------|------|
| **资源概览** | CPU、内存、NPU 配额和使用情况 |
| **活跃环境** | 当前运行中的开发环境 |
| **活跃任务** | 当前运行中的训练任务 |
| **系统公告** | 管理员发布的公告信息 |

## 资源概览

### 查看配额

资源概览卡片显示：

- **CPU**：已用/总量，使用率百分比
- **内存**：已用/总量，使用率百分比
- **NPU**：已用/总量，使用率百分比

### 配额状态

| 状态 | 颜色 | 说明 |
|------|------|------|
| 正常 | 🟢 绿色 | 使用率 < 70% |
| 警告 | 🟡 黄色 | 使用率 70-90% |
| 紧张 | 🔴 红色 | 使用率 > 90% |

## 快捷操作

仪表盘提供以下快捷入口：

1. **创建环境**：快速创建新的开发环境
2. **创建任务**：快速创建新的训练任务
3. **查看详情**：进入详细的使用统计页面

## 活跃资源

### 开发环境

显示运行中的开发环境列表：

| 信息 | 说明 |
|------|------|
| 名称 | 环境名称 |
| 状态 | 运行状态 |
| 资源 | CPU/内存配置 |
| 运行时长 | 已运行时间 |

点击环境名称可进入详情页。

### 训练任务

显示运行中的训练任务列表：

| 信息 | 说明 |
|------|------|
| 名称 | 任务名称 |
| 状态 | 运行状态 |
| 资源 | CPU/内存/NPU 配置 |
| 进度 | 训练进度（如有） |

## 使用建议

1. **定期检查配额**：关注资源使用情况，避免超限
2. **及时释放资源**：停止不使用的环境
3. **关注系统公告**：了解平台动态和维护通知
