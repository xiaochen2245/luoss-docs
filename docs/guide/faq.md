# 常见问题

## 环境相关

### Q: Code Server 打开后显示密码错误？

**A**: 请在环境详情页面查看正确的访问密码。

### Q: 环境启动失败怎么办？

**A**: 检查以下内容：
1. 配额是否充足（查看仪表盘）
2. 资源配置是否在允许范围内
3. 查看环境详情的错误信息
4. 联系管理员获取帮助

### Q: 环境自动停止了？

**A**: 系统会在环境空闲超过设定时间后自动停止。如需继续使用：
1. 点击 **启动** 按钮重新启动
2. 考虑增加自动停止时间
3. 定期访问环境更新访问时间

### Q: 删除环境后数据还在吗？

**A**: 是的。用户存储独立于环境，删除环境不会删除存储中的文件。

## 训练任务相关

### Q: 任务一直处于 Pending 状态？

**A**: 可能原因：
1. **集群资源不足** - 等待其他任务释放资源
2. **优先级较低** - 被高优先级任务抢占
3. **NPU 资源不足** - 检查 NPU 配额和使用情况

解决方案：
- 降低资源配置
- 提高任务优先级
- 等待资源释放

### Q: 训练任务失败如何排查？

**A**:
1. 进入任务详情页
2. 查看 **事件** 了解失败原因
3. 查看 **日志** 检查运行日志
4. 常见错误：
   - OOM（内存不足）：增加内存配置
   - 镜像拉取失败：检查镜像地址
   - 存储挂载失败：检查存储配额

### Q: 如何查看训练输出？

**A**: 训练输出保存在 `/models/output` 目录：
1. 在开发环境中打开此目录
2. 或通过文件管理功能下载

## 存储相关

### Q: 存储空间不足怎么办？

**A**:
1. 清理不需要的文件
2. 删除旧的模型检查点
3. 联系管理员增加配额

### Q: 如何在不同环境间共享文件？

**A**: 所有环境共享同一个用户存储，直接访问对应路径即可。

### Q: 如何备份数据？

**A**:
```bash
# 打包重要文件
cd /config/workspace
tar -czvf backup.tar.gz projects/

# 在 Code Server 中下载备份文件
```

---

## 配额相关

### Q: 如何查看我的配额？

**A**:
1. 查看仪表盘的配额概览
2. 进入 **使用统计** 页面查看详情

### Q: 配额不足怎么办？

**A**:
1. 停止不使用的环境
2. 删除已完成的训练任务
3. 清理存储空间
4. 联系管理员申请增加配额

### Q: NPU 配额为什么是 0？

**A**: 默认用户可能没有 NPU 配额。请联系管理员分配。

---

## 账户相关

### Q: 忘记密码怎么办？

**A**: 联系管理员重置密码。

### Q: 如何修改密码？

**A**:
1. 点击左侧菜单 **设置**
2. 点击 **修改密码**
3. 输入旧密码和新密码

## 其他问题

### Q: 平台支持哪些开发模板？

**A**: 平台提供多种预配置模板：

| 模板 | 说明 |
|------|------|
| MindIE | 华为昇腾推理开发 |
| MindSpeed | LLM 训练和推理 |
| vLLM | 高性能 LLM 推理 |
| MindSpore | MindSpore 框架开发 |

### Q: 如何联系技术支持？

**A**:
- 联系平台管理员
- 查看系统公告获取联系方式

### Q: 如何报告 Bug？

**A**:
1. 记录问题发生的时间和操作步骤
2. 截取错误信息截图
3. 联系管理员
