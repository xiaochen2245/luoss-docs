# 设置

设置页面用于管理您的账户信息和偏好设置。

## 功能概述

| 功能 | 说明 |
|------|------|
| **个人信息** | 查看和修改个人信息 |
| **修改密码** | 更改登录密码 |
| **偏好设置** | 自定义界面设置 |

## 个人信息

### 查看信息

| 字段 | 说明 |
|------|------|
| 用户名 | 登录用户名（不可修改） |
| 邮箱 | 注册邮箱 |
| 角色 | 用户角色 |
| 命名空间 | Kubernetes 命名空间 |

### 修改信息

1. 点击 **编辑** 按钮
2. 修改邮箱等信息
3. 点击 **保存**

## 修改密码

### 步骤

1. 点击 **修改密码**
2. 输入旧密码
3. 输入新密码
4. 确认新密码
5. 点击 **确认**

::: warning 安全提示
- 使用强密码
- 定期更换密码
- 不要与他人共享密码
:::

## 偏好设置

| 设置 | 说明 |
|------|------|
| 主题 | 浅色/深色主题 |
| 语言 | 界面语言 |

## 常见问题

### Q: 忘记密码怎么办？

A: 联系管理员重置密码。

### Q: 用户名可以修改吗？

A: 用户名创建后不可修改。
