# 设置

设置页面用于管理您的账户信息、安全设置和查看审计记录。

## 功能概述

| 功能 | 说明 |
|------|------|
| **个人信息** | 查看个人信息和账户状态 |
| **安全设置** | 修改登录密码 |
| **审计记录** | 查看账户操作审计日志 |
| **系统配置** | 训练任务运行时长限制（仅管理员可见） |

## 个人信息

### 查看信息

| 字段 | 说明 |
|------|------|
| 用户名 | 登录用户名（不可修改） |
| 邮箱 | 注册邮箱 |
| 用户 ID | 系统分配的唯一标识 |
| 注册时间 | 账户创建时间 |
| 账户权限 | 平台管理员 / 普通用户 |
| 账户状态 | 已激活 |

## 安全设置

### 修改密码

1. 切换到 **安全设置** 标签页
2. 输入当前密码
3. 输入新密码（至少 6 个字符）
4. 确认新密码
5. 点击 **确认**

::: warning 安全提示
- 使用强密码
- 定期更换密码
- 不要与他人共享密码
:::

## 审计记录

审计记录标签页显示您账户的所有操作日志。

### 支持的操作类型

| 操作 | 说明 |
|------|------|
| 登录 / 登出 | 账户登录和登出记录 |
| 创建环境 | 创建开发环境 |
| 启动 / 停止环境 | 环境启停操作 |
| 删除环境 | 删除开发环境 |
| 创建训练任务 | 提交训练任务 |
| 启动 / 停止训练 | 训练任务启停 |
| 删除训练任务 | 删除训练任务 |
| 修改密码 | 密码变更记录 |

### 筛选功能

- **操作类型**：按操作类型筛选
- **状态**：成功 / 失败
- **时间范围**：选择日期和时间区间

### 审计字段

| 字段 | 说明 |
|------|------|
| 时间 | 操作发生的时间 |
| 操作 | 执行的操作类型 |
| 详情 | 操作的详细信息 |
| IP 地址 | 操作来源 IP |
| 状态 | 成功或失败 |

## 系统配置（仅管理员）

管理员可以在系统配置标签页设置训练任务的全局限制：

| 配置项 | 说明 | 范围 |
|--------|------|------|
| 全局最大运行时长 | 训练任务的最大运行时间（分钟） | 1 - 10080 分钟（1-7 天） |
| 超时检查间隔 | 系统检查超时任务的时间间隔 | 30 - 3600 秒 |
| 自动终止 | 是否自动终止超时的训练任务 | 开 / 关 |

## 常见问题

### Q: 忘记密码怎么办？

A: 联系管理员重置密码。

### Q: 用户名可以修改吗？

A: 用户名创建后不可修改。
