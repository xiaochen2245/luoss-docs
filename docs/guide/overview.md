# 概述

欢迎使用 LuoSS 络书算衡 - K8s 租户管理平台。

## 平台简介

LuoSS 是一个基于 Kubernetes 的多租户管理平台，提供：

- **云端开发环境**：基于 Code Server 的在线 IDE
- **训练任务管理**：支持单机和分布式训练
- **资源配额管理**：CPU、内存、NPU 资源管理
- **多租户隔离**：每个用户独立命名空间

## 用户界面

### 导航菜单

| 菜单 | 说明 |
|------|------|
| 仪表盘 | 资源概览和快捷操作 |
| 开发环境 | 管理云端开发环境 |
| 训练任务 | 管理训练任务 |
| 数据存储 | 查看存储使用情况 |
| 镜像存储 | 查看可用镜像 |
| 我的作业 | 查看提交的作业 |
| 使用统计 | 查看资源使用统计 |
| 设置 | 账户设置 |

## 核心概念

### 命名空间

每个用户有独立的 Kubernetes 命名空间 `user-{username}`，实现资源隔离。

### 配额

每个用户有资源配额限制：

| 资源 | 说明 |
|------|------|
| CPU | CPU 核数 |
| 内存 | 内存大小 (Gi) |
| NPU | 神经网络处理器数量 |

### 存储

用户存储自动挂载到环境：

| 环境 | 挂载路径 |
|------|----------|
| 开发环境 | `/config/workspace` |
| 训练任务 | `/models` |
