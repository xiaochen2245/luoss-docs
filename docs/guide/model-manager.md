# 数据存储

数据存储页面展示您的用户存储使用情况。

## 功能概述

| 功能 | 说明 |
|------|------|
| **存储概览** | 查看存储配额和使用情况 |
| **文件管理** | 浏览和管理存储文件 |
| **数据上传** | 上传数据到存储 |

## 存储概览

### 查看配额

| 信息 | 说明 |
|------|------|
| 已用空间 | 当前使用的存储空间 |
| 总配额 | 分配的存储配额 |
| 使用率 | 已用/总量百分比 |

### 存储路径

用户存储挂载到开发环境和训练任务：

| 环境 | 挂载路径 |
|------|----------|
| 开发环境 | `/config/workspace` |
| 训练任务 | `/models` |

::: tip 共享存储
所有环境和任务共享同一个用户存储，数据在不同环境间互通。
:::

## 目录结构

推荐的目录结构：

```
/config/workspace/     # 开发环境
/models/               # 训练任务
├── datasets/          # 数据集
│   ├── train/         # 训练数据
│   └── val/           # 验证数据
├── models/            # 预训练模型
│   └── llama-3-8b/
├── output/            # 训练输出
│   └── exp-001/
└── workspace/         # 工作目录
```

## 使用建议

1. **分类存放**：按类型组织数据
2. **定期清理**：删除不需要的文件
3. **备份重要数据**：定期下载备份

## 常见问题

### Q: 存储空间不足？

A: 清理不需要的文件，或联系管理员增加配额。

### Q: 如何在不同环境间共享文件？

A: 所有环境共享同一存储，直接访问对应路径即可。
