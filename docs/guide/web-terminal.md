# Web 终端

Web 终端功能允许您直接在浏览器中连接到训练任务的容器，方便调试和排查问题。

<FeatureBadge status="stable" />

## 功能概述

| 功能 | 说明 |
|------|------|
| **实时连接** | 通过 WebSocket 实时连接容器终端 |
| **交互式操作** | 支持完整的终端交互操作 |
| **窗口调整** | 支持终端窗口大小自适应 |

## 使用 Web 终端

### 通过训练任务

1. 进入 **训练任务** 页面
2. 找到运行中的任务
3. 点击 **终端** 按钮
4. 系统自动连接到任务容器

### 通过管理后台

管理员可以通过 **实时作业** 页面直接连接任何运行中的任务容器。

## 终端操作

### 基本操作

连接成功后，您可以像使用普通终端一样操作：

```bash
# 查看当前目录
ls -la

# 查看训练进程
ps aux | grep python

# 查看 GPU/NPU 使用情况
npu-smi info

# 查看训练日志
tail -f /models/output/train.log
```

### 常用调试命令

```bash
# 检查环境变量
env | grep -i rank

# 查看资源使用
top -bn1

# 检查网络连通性
ping -c 3 $MASTER_ADDR

# 查看磁盘使用
df -h /models
```

## 注意事项

::: warning 注意事项
1. 终端连接与容器生命周期绑定，容器重启后连接会断开
2. 终端中的操作直接影响运行中的容器，请谨慎操作
3. 不建议通过终端修改训练代码或配置，可能导致任务异常
:::

## 常见问题

### Q: 终端连接失败？

A: 请检查：
1. 任务是否处于 Running 状态
2. 网络连接是否正常
3. 浏览器是否支持 WebSocket

### Q: 终端显示乱码？

A: 尝试调整终端编码设置，或在终端中执行 `export LANG=en_US.UTF-8`。

### Q: 终端操作会影响训练任务吗？

A: 正常的查看操作不会影响训练。但请避免终止关键进程或修改训练配置文件。
