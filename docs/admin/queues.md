# 队列管理

队列管理页面用于管理 Volcano 调度队列。

## 功能概述

| 功能 | 说明 |
|------|------|
| **队列列表** | 查看所有队列 |
| **队列详情** | 查看队列配置 |
| **创建/编辑** | 创建和编辑队列 |
| **删除队列** | 删除不需要的队列 |

## 队列列表

| 字段 | 说明 |
|------|------|
| 队列名称 | 队列标识 |
| 状态 | 队列状态 |
| 权重 | 调度权重 |
| 能力 | 资源能力 |
| 已分配 | 已分配资源 |

## 队列层级关系

系统支持层级队列管理：

```
root
├── debug (调试池)
│   └── debug-sub
└── compute (计算池)
    ├── user-alice-compute
    ├── user-bob-compute
    └── ...
```

### 父子队列约束

::: warning 重要约束
1. **子队列资源不能超过父队列**：子队列的 Deserved 和 Capability 不能超过父队列的限制
2. **所有子队列资源总和不能超过父队列**：多个子队列的资源配额总和不能超过父队列的配额
3. **父队列必须存在**：设置父队列前，请确保父队列已创建
:::

## 创建/编辑队列

### 基础配置

| 字段 | 说明 | 限制 |
|------|------|------|
| 队列名称 | 队列唯一标识 | 创建后不可修改 |
| 父队列 | 父队列名称 | 创建后不可修改 |
| 权重 | 调度权重 | 1-65535 |
| 优先级 | 队列优先级 | 0-1000000 |
| 可回收 | 资源是否可被回收 | 开启/关闭 |

### 资源配额

| 资源 | 格式 | 示例 |
|------|------|------|
| CPU | 核数 | 8, 16, 32 |
| 内存 | Gi | 32Gi, 64Gi |
| NPU | 数量 | 8, 16, 32 |

### 常见错误

#### 队列配置冲突

当出现以下情况时会显示冲突错误：

| 错误原因 | 解决方法 |
|----------|----------|
| 子队列资源超过父队列 | 降低子队列资源配额，或增加父队列配额 |
| 所有子队列总和超过父队列 | 重新分配各子队列的资源配额 |
| 父队列不存在 | 先创建父队列，再创建子队列 |

#### 队列配置无效

| 错误原因 | 解决方法 |
|----------|----------|
| 资源格式错误 | 使用正确的格式（如 32Gi） |
| 父队列名称错误 | 确认父队列名称正确 |
| 权重/优先级超范围 | 调整到有效范围内 |
