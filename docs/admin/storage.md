# 存储管理

存储管理页面用于管理用户的存储配额和使用情况。

## 功能概述

| 功能 | 说明 |
|------|------|
| **存储列表** | 查看所有用户存储 |
| **配额管理** | 设置存储配额 |
| **使用统计** | 存储使用情况 |

## 存储列表

| 字段 | 说明 |
|------|------|
| 用户名 | 用户标识 |
| PVC 名称 | 持久卷声明名称 |
| 配额 | 存储配额 |
| 已用 | 已使用空间 |
| 使用率 | 使用百分比 |
| 状态 | 存储状态 |

## 存储配额

### 设置配额

1. 选择用户
2. 点击 **设置配额**
3. 输入存储大小（Gi）
4. 保存设置

### 配额调整

| 操作 | 说明 |
|------|------|
| 增加配额 | 需存储类支持扩容 |
| 减少配额 | 不支持，需删除重建 |

## 使用统计

### 使用概览

| 指标 | 说明 |
|------|------|
| 总存储 | 集群存储总量 |
| 已分配 | 已分配存储 |
| 可用 | 可用存储 |

## 存储操作

### 清理存储

如需清理用户存储：

1. 选择用户
2. 点击 **清理**
3. 确认操作

::: warning 警告
清理操作会删除存储中的所有数据，不可恢复。
:::
