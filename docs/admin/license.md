# 许可管理

本文档介绍平台的许可证管理系统，包括许可证激活、状态查看和指纹绑定等功能。

## 概述

LuoSS 平台采用许可证（License）机制控制平台使用规模。许可证定义了集群可使用的最大资源限制：

| 限制项 | 说明 |
|--------|------|
| 最大节点数 | 集群允许的最大 Kubernetes 节点数 |
| 最大 NPU 数 | 集群允许的最大 NPU 总数 |
| 最大用户数 | 平台允许注册的最大用户数 |

::: warning 许可证过期
许可证过期后，平台将拒绝所有新的 API 请求（健康检查和许可证管理接口除外）。请务必在许可证过期前完成续期。
:::

## 查看许可证信息

进入 **管理后台** → **许可管理**，查看当前许可证状态。

### 许可证详情

| 字段 | 说明 |
|------|------|
| 许可证 ID | 唯一标识符 |
| 客户名称 | 授权客户名称 |
| 签发时间 | 许可证签发日期 |
| 过期时间 | 许可证有效截止日期 |
| 剩余天数 | 距离过期还剩多少天 |
| 有效状态 | 是否有效 |

### 使用情况

| 字段 | 说明 |
|------|------|
| 当前节点数 | 集群当前节点数量 |
| 当前 NPU 数 | 集群当前 NPU 总量 |
| 当前用户数 | 平台已注册用户数 |

### 通过 API 查看

```bash
GET /api/v1/admin/license
Authorization: Bearer <admin-token>
```

响应：

## 激活许可证

### 获取激活码

联系平台供应商获取激活码。激活码与集群 CA 证书指纹绑定，不可跨集群使用。

### 激活步骤

1. 进入 **许可管理** 页面
2. 输入激活码
3. 点击 **激活**

### 通过 API 激活

```bash
POST /api/v1/admin/license/activate
Authorization: Bearer <admin-token>
Content-Type: application/json

{
  "activation_code": "XXXX-XXXX-XXXX-XXXX"
}
```

::: tip 指纹绑定
激活码会绑定到当前集群的 CA 证书指纹。如果集群证书变更（如重新初始化集群），需要重新申请激活码。
:::

## 集群指纹

集群指纹是集群 CA 证书的 SHA256 哈希值，用于标识唯一的 Kubernetes 集群。

### 查看集群指纹

```bash
GET /api/v1/admin/license/fingerprint
Authorization: Bearer <admin-token>
```

响应：

```json
{
  "fingerprint": "sha256:a1b2c3d4e5f6..."
}
```

### 指纹用途

- 申请许可证时，需要提供集群指纹
- 激活码与指纹绑定，防止许可证被转移到其他集群
- 集群证书变更后指纹会改变，需重新申请

## 许可证过期处理

### 过期前提醒

许可证在过期前 30 天，系统会通过以下方式提醒：

- API 响应头中添加 `X-License-Warning` 警告信息
- 管理后台显示过期提醒
- 站内信通知管理员

### 过期后影响

| 功能 | 状态 |
|------|------|
| 健康检查接口 | ✅ 正常 |
| 登录认证 | ✅ 正常 |
| 许可证管理接口 | ✅ 正常 |
| 其他所有 API | ❌ 拒绝请求 |

### 续期步骤

1. 获取当前集群指纹
2. 将指纹发送给供应商获取新激活码
3. 在许可管理页面输入新激活码激活

## 常见问题

### Q: 如何判断许可证是否即将过期？

A: 登录管理后台，进入 **许可管理** 页面查看剩余天数。许可证在过期前 30 天会显示警告提示。

### Q: 许可证过期后数据会丢失吗？

A: 不会。许可证过期只是限制 API 访问，数据库和 Kubernetes 资源不受影响。续期后即可恢复正常使用。

### Q: 集群证书更换后许可证失效怎么办？

A: 需要使用新集群指纹重新申请激活码。执行步骤：
1. 查看新集群指纹
2. 联系供应商获取新激活码
3. 在许可管理页面激活

### Q: 使用量超过许可证限制会怎样？

A: 已有的资源不受影响，但新用户注册或新节点加入可能会被限制。建议在接近限制时联系供应商升级许可证。
