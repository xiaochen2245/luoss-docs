# 配额管理

配额管理页面用于设置和管理用户的资源配额。

## 功能概述

| 功能 | 说明 |
|------|------|
| **配额列表** | 查看所有用户配额 |
| **设置配额** | 修改用户配额 |
| **使用统计** | 查看配额使用情况 |

## 配额列表

| 字段 | 说明 |
|------|------|
| 用户名 | 用户标识 |
| CPU 配额 | CPU 总配额 |
| CPU 已用 | 已使用 CPU |
| 内存配额 | 内存总配额 |
| 内存已用 | 已使用内存 |
| NPU 配额 | NPU 总配额 |
| NPU 已用 | 已使用 NPU |

## 设置配额

### 步骤

1. 找到目标用户
2. 点击 **设置配额**
3. 修改资源值：

| 资源 | 单位 | 说明 |
|------|------|------|
| CPU | 核 | CPU 配额上限 |
| 内存 | Gi | 内存配额上限 |
| NPU | 卡 | NPU 配额上限 |

4. 点击 **保存**

### 配额更新

- 增加配额：立即生效
- 减少配额：仅影响新资源

## 配额策略

### 推荐配置

| 用户类型 | CPU | 内存 | NPU |
|----------|-----|------|-----|
| 普通用户 | 4 | 8 | 0 |
| 开发人员 | 8 | 16 | 2 |
| 算法工程师 | 16 | 32 | 4-8 |

### 动态调整

根据使用情况动态调整：

1. 定期检查使用率
2. 为活跃用户增加配额
3. 回收长期不用的配额
