# 用户管理

用户管理页面用于创建、编辑和管理平台用户。

## 功能概述

| 功能 | 说明 |
|------|------|
| **用户列表** | 查看所有用户 |
| **创建用户** | 添加新用户 |
| **编辑用户** | 修改用户信息 |
| **配额管理** | 设置用户配额 |
| **删除用户** | 删除用户账户 |

## 用户列表

| 字段 | 说明 |
|------|------|
| 用户名 | 登录用户名 |
| 邮箱 | 用户邮箱 |
| 角色 | 普通用户/管理员 |
| 状态 | 启用/禁用 |
| 创建时间 | 账户创建时间 |

## 创建用户

### 步骤

1. 点击 **添加用户** 按钮
2. 填写信息：

| 字段 | 必填 | 说明 |
|------|------|------|
| 用户名 | 是 | 唯一标识符 |
| 密码 | 是 | 初始密码 |
| 邮箱 | 否 | 用于通知 |
| 管理员 | 否 | 是否为管理员 |

3. 点击 **创建**

### 命名空间创建

创建用户时自动：

1. 创建 Kubernetes 命名空间 `user-{username}`
2. 创建默认 ResourceQuota
3. 初始化用户存储 PVC

## 编辑用户

### 修改信息

1. 点击 **编辑** 按钮
2. 修改邮箱或角色
3. 保存更改

### 重置密码

1. 点击 **重置密码**
2. 输入新密码
3. 确认重置

## 配额管理

### 设置配额

1. 点击 **设置配额**
2. 配置资源：

| 资源 | 单位 | 说明 |
|------|------|------|
| CPU | 核 | CPU 配额 |
| 内存 | Gi | 内存配额 |
| NPU | 卡 | NPU 配额 |

3. 保存设置

### 配额模板

| 用户类型 | CPU | 内存 | NPU |
|----------|-----|------|-----|
| 普通用户 | 4 | 8 Gi | 0 |
| 开发人员 | 8 | 16 Gi | 2 |
| 算法工程师 | 16 | 32 Gi | 4 |

## 禁用用户

1. 点击 **禁用** 按钮
2. 确认操作

禁用后：
- 用户无法登录
- 已运行资源继续运行

## 删除用户

::: danger 危险操作
删除用户会：
1. 删除用户数据库记录
2. 删除 Kubernetes 命名空间
3. 删除所有资源和数据
4. **操作不可恢复！**
:::

### 删除步骤

1. 确认用户已停止所有资源
2. 点击 **删除**
3. 输入用户名确认
4. 执行删除
